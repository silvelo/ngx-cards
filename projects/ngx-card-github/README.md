# NgxCardGithub

Angular component to display your github profile.

## Usage

```
import { NgxCardGithubModule } from "NgxCardGithubModule"
...
imports: [
  NgxCardGithubModule
]
...
```

In your html use the component

```
<ngx-card-github userId="YOUR GITHUB ID OR USERNAME"></ngx-card-github>

```
